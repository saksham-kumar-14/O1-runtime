# O1 runtime
`O1` is a lightweight runtime built in Go

## Features
- Lightweight
- Overlay Filesystem
- Network Bridge: now my laptop can communicate with the containers running
- `o1 ps`: list of running containers
- `o1 stop <CONTAINER_ID>`: stop the container
- [ ] `o1 exec`
- [ ] Dynamic Networking
- [ ] `o1 logs`
